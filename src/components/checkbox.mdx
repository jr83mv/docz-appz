# CheckBox

The CheckBox component allows users to select or deselect options in a list of choices. It provides a customizable and accessible interface to indicate selection status.

### Best practices

- Use checkboxes when users can select one or multiple options from a list.
- Clearly label the checkbox to indicate what option it represents.
- Ensure checkboxes have appropriate spacing and are easy to tap on mobile devices.
